# 一、拓扑 & 网格合法性指标（准入类，必检）
这类指标判断模型**是否为有效可用网格**，是模型上线/仿真/渲染的基础门槛。

## 1. 流形网格 Manifold Mesh
- **释义**：三维建模通用标准网格，要求**每条边最多被2个三角面共享**、每个顶点邻域连续无分叉。
- 细分问题项：
  - **非流形边 Non-manifold Edge**：一条边被 ≥3 个面共用，属于严重拓扑错误，会导致切片、布尔运算、渲染报错。
  - **非流形顶点 Non-manifold Vertex**：顶点连接的面/边形成断裂、分叉拓扑。
- **判定标准**：合格模型 = 无非流形边、无非流形顶点。
- **对应接口**：`mrmeshpy.findNonManifoldEdges()`

## 2. 退化面 Degenerate Face
- **释义**：面积无限趋近于0的三角面，由三个顶点共线/两点重合导致。
- 表现：三角面压扁成线段/点，计算时会出现除零、法向异常。
- **判定标准**：合格模型 = 退化面数量为 0。
- **对应接口**：`mrmeshpy.findDegenerateFaces()`

## 3. 孔洞 Hole
- **释义**：网格表面缺失面形成的镂空区域。
  - 封闭模型（实体模型）：**孔洞数必须为 0**；
  - 开放面片（如布料、薄板）：允许存在边界孔洞。
- **对应接口**：`mrmeshpy.findHoles()`

## 4. 重复元素 Duplicate Vertices/Faces
- **重复顶点**：空间坐标完全一致的多个顶点，造成网格冗余、表面抖动。
- **重复三角面**：三个顶点索引完全一致的面，重复叠加无意义。
- **判定**：合格模型无重复顶点、无重复面。

## 5. 边界边 Boundary Edge
- **释义**：仅被**1个三角面**占用的边，是网格的轮廓边缘。
- 用途：判断模型是否封闭
  - 实体封闭模型：边界边总数 = 0
  - 开放面片：边界边构成完整外轮廓

## 6. 法向一致性 Normal Consistency
- **释义**：三角面法向统一朝向模型外侧/内侧。
- 问题：法向反转、朝向混乱，会导致渲染发黑、光照错误、体积计算异常。
- **判定**：全局法向方向统一，无大面积反转面。
- **对应接口**：`mrmeshpy.checkNormals()`

# 二、网格质量指标（优化类，影响计算/渲染稳定性）
针对三角面形态、分布均匀度，主要用于**重拓扑、网格简化、仿真计算**评测。

## 1. 面长宽比 Aspect Ratio
- **释义**：三角面最长边与最短边的比值，衡量三角形畸变程度。
  - 理想等边三角形：比值 = 1（最优）
  - 数值越大，三角面越狭长，质量越差
- **通用阈值**
  - 工业通用：≤ 5 优秀；≤ 10 合格；＞10 劣质面
- 统计值：整网取**最大值、平均值、95分位值**

## 2. 内角 Angle of Triangle
- **释义**：单个三角面三个内角的大小。
- 问题：存在**极小角（接近0°）**或**极大角（接近180°）**，等同于退化面，仿真、插值极易出错。
- **通用阈值**：单三角面内角建议 15° ~ 150° 之间。

## 3. 面面积 Face Area
- **释义**：每个三角面的空间面积。
- 统计维度：
  - 平均面积、最大/最小面积
  - 面积标准差：标准差越小 → 网格分布越均匀
- 用途：评估网格细分是否均匀，局部过密/过疏问题。

## 4. 二面角 Dihedral Angle
- **释义**：**相邻两个三角面**之间的夹角（面与面的转折角度）。
- 用途：
  1. 检测模型尖锐棱角、平滑过渡区域；
  2. 网格光顺、平滑算法评测（平滑后二面角变化越小，细节保留越好）。

## 5. 顶点度数 Vertex Valence
- **释义**：单个顶点连接的三角面/边数量。
- 理想三角网格：顶点度数集中在 5~6 之间，分布越集中，网格越规整。

# 三、几何相似度指标（比对类，有GT标准模型时核心指标）
用于**模型重建、扫描配准、网格修复、模型简化**，量化被测模型与真值(GT)的几何误差。

## 1. 豪斯多夫距离 Hausdorff Distance（行业金标）
### 双向 Hausdorff（最常用）
- **释义**：衡量两个网格**整体最大偏差**，描述“一个模型上最远点到另一个模型的最短距离”。
  - 单向：A→B、B→A 分别计算
  - 双向 Hausdorff = 两个单向距离的**最大值**
- **物理意义**：两个模型之间**最大几何偏差**，直观反映极限误差。
- **优缺点**：对离群点、局部异常极度敏感。
- **使用建议**：
  - 单位统一（mm/m），提前做坐标对齐；
  - 大模型用 `mrcudapy` GPU 加速计算。

## 2. 平均距离 Mean Distance / MAE
- **释义**：模型所有采样点到GT网格的**最短距离平均值**（平均绝对误差）。
- **物理意义**：整体平均偏差，反映模型全局拟合精度，不受个别异常点过度影响。

## 3. 分位距离 Percentile Distance
- **常用**：90%、95%、99% 分位距离
- **释义**：剔除极端离群点后，大部分区域的误差水平。
- **场景**：工业评测首选（实际工程中允许极个别微小异常点），比单纯最大值更客观。

## 4. 均方误差 MSE / 均方根误差 RMSE
- **MSE**：所有距离的平方取平均，放大大误差的权重，侧重惩罚局部大偏差。
- **RMSE**：MSE 开根号，单位和原始距离一致，直观易读。
- 适用：对局部大误差零容忍的场景（如精密零件）。

## 5. 法向误差 Normal Error
- **释义**：被测模型面法向 与 GT 模型对应区域法向的**夹角误差（角度/弧度）**。
- 用途：评测表面**曲率、光影、细节**还原度，对外观类模型至关重要。
- 单位：°（角度），阈值一般＜5°为优秀。

## 6. 体积偏差 Volume Deviation
- **释义**：被测模型体积 与 GT 体积的差值/相对误差。
- 公式：`相对体积误差 = |V_test - V_gt| / V_gt × 100%`
- 适用：实体零件、3D打印、仿真模型评测。

## 7. 表面积偏差 Surface Area Deviation
- **释义**：两个模型表面积的相对误差，侧重表面轮廓、薄壳类模型。

# 四、网格轻量化/效率指标（简化、压缩场景专用）
针对网格减面、轻量化后的综合评测。

## 1. 面数/顶点数 Face Count / Vertex Count
- 基础统计：总三角面数、总顶点数。

## 2. 压缩率 Compression Ratio
- **公式**：
  `压缩率 = (原始面数 - 简化后面数) / 原始面数 × 100%`
- 含义：面数减少比例，结合几何误差做**“压缩率-精度”权衡**。

## 3. 网格密度 Mesh Density
- 单位空间内的面/顶点数量，衡量局部疏密程度。

# 补充通用规则
1. **单位统一**：所有模型坐标单位必须一致（mm/m），距离类指标才有效。
2. **前置对齐**：比对GT前，必须做刚体配准（平移+旋转），否则误差无参考意义。
3. **指标组合使用**：
   - 只看最大值（Hausdorff）：关注极限缺陷
   - 只看平均值：关注整体效果
   - 工程最优：**95分位距离 + 平均距离 + 最大距离** 三者结合。
